# TimberRunSteamGame

TimberRunSteamGame is an Unreal Engine project built around a stylized forest environment and an interactive tree chopping gameplay loop.

Project update date: March 19, 2026

Total development time so far: About 4 hours

Recent updates include:

- Added a forest environment using assets from the Fab library.
- Added a chop mechanic for player tree interactions.
- Added tree health so trees can take multiple hits before breaking.
- Added tree destruction behavior so a tree is destroyed after 5 chops.
